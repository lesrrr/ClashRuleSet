## 下载后需要配置的几个参数：
```
check_interval = 300  # 检测时间间隔，单位：秒  可自由调整，如果网络较差可适当延长检测时间
workers = 12  # 根据自己的设备设置进程数量
code = ""  # 粘贴code
```
